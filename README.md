# Cotizador RanchGPT

Este proyecto es una herramienta interactiva para cotizar los servicios de **RanchGPT**, un asistente inteligente diseñado para la gestión y optimización de feedlots y ganadería.

## Archivos del Proyecto

- `cotizador_ranchgpt_v3.html`: Interfaz web interactiva del cotizador. Permite ingresar parámetros del feedlot y obtener una cotización estimada basada en diferentes planes.
