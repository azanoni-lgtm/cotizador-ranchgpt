# Cotizador RanchGPT

Este proyecto es una herramienta interactiva para cotizar los servicios de **RanchGPT**, un asistente inteligente diseñado para la gestión y optimización de feedlots y ganadería.

## Acceso Web

Puedes acceder a la versión publicada del cotizador directamente en el siguiente enlace:
👉 **[https://azanoni-lgtm.github.io/cotizador-ranchgpt/](https://azanoni-lgtm.github.io/cotizador-ranchgpt/)**

## Estructura del Proyecto

- `index.html`: Interfaz web interactiva del cotizador. Permite ingresar parámetros del feedlot, estimar precios y sincronizar prospectos directamente con HubSpot.
- `.gitignore`: Configuración para evitar subir archivos innecesarios.
